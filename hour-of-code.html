Hour of Code

What is the Hour of Code?

The Hour of Code is a global movement that introduces students of all ages to the basics of computer science. With fun, one-hour coding activities, it aims to demystify coding and show that anyone can learn the fundamentals, sparking interest and curiosity in the tech world.

How We Organized It

Our team planned and facilitated the Hour of Code event for school students, creating engaging activities that introduced them to the basics of computer science. We provided personalized guidance and support, creating an inclusive and interactive learning environment that encouraged students to explore coding and develop their tech skills.
